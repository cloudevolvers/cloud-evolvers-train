Buyer security answers

Security questionnaire

Use this page for first-pass vendor review of Cloud Evolvers training, readiness scans, and Microsoft cloud consulting. It is not a certification or legal opinion; it is a public summary of operating practices and the evidence topics we can discuss during a scoped review.

Service scope

Cloud Evolvers provides MCT-led Microsoft Azure and Microsoft 365 training, readiness scans, free cloud tools, and implementation workshops through Spot Cloud B.V.

Data collected

Typical data includes contact details submitted through forms, training request context, selected scanner answers, and privacy-safe page/tool analytics.

Compliance mapping

Readiness work can map gaps to Microsoft Secure Score, Defender for Cloud, Purview, Entra ID, backup, supplier, DORA, NIS2, NIST, and CIS themes.

Who operates Cloud Evolvers?

Cloud Evolvers is operated by Spot Cloud B.V. in the Netherlands. Public company details are listed in the footer and trust center.

Do free tools require sensitive tenant access?

No. Public tools are designed for directional checks and training guidance. The Microsoft cloud compliance readiness scanner uses questionnaire input and does not require tenant admin consent or secret upload.

How are consultation requests handled?

Submitted forms are used for follow-up and scoping. Requests should avoid passwords, private keys, production secrets, export-controlled data, and regulated personal data unless a separate agreement and secure channel are in place.

What operational evidence can be discussed?

For scoped engagements, evidence topics can include identity posture, privileged access, backup coverage, logging, endpoint management, Defender and Purview status, supplier register hygiene, and remediation ownership.

Does Cloud Evolvers certify DORA, NIST, CIS, or NIS2 compliance?

No. Cloud Evolvers can help teams understand gaps, prepare evidence, and plan remediation work. Formal assurance, legal interpretation, and certification remain with the customer's auditors, legal counsel, and accredited assessors.

How can security issues be reported?

Security issues can be sent to training@cloudevolvers.com. Availability information is published on the status page.
