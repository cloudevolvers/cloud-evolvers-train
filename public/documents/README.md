# Terms of Service Documents

This directory contains the Terms of Service documents for Cloud Evolvers (Spot Cloud B.V.).

## Files

- **Algemene voorwaardenSpotCloud.pdf** - Original Dutch version of the Terms of Service
- **terms-of-service-en.html** - English version (HTML format, can be printed to PDF)
- **terms-of-service-en-full.md** - English version (Markdown format)

## How to Generate PDF from HTML

The English HTML version can be converted to PDF in multiple ways:

### Option 1: Browser Print to PDF
1. Open `terms-of-service-en.html` in any web browser
2. Press Ctrl+P (Windows/Linux) or Cmd+P (Mac)
3. Select "Save as PDF" as the destination
4. Save as `terms-of-service-en.pdf`

### Option 2: Using wkhtmltopdf (if installed)
```bash
wkhtmltopdf terms-of-service-en.html terms-of-service-en.pdf
```

### Option 3: Using Chromium/Chrome headless
```bash
chromium --headless --print-to-pdf=terms-of-service-en.pdf terms-of-service-en.html
```

## Language Availability

Users can download the Terms of Service in the following languages on the website:

- **Nederlands (Dutch)** - Original PDF document
- **English** - HTML version (easily printable to PDF)
