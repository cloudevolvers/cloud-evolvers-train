GPT-5.4 on Azure OpenAI: The Full GPT-5 Model Family Explained

Yair Knijn · March 19, 2026 · 10 min read
Azure OpenAI GPT-5.4 Reasoning Models

GPT-5 landed on Azure OpenAI in August 2025. Seven months and twenty-something model variants later, we are at GPT-5.4. Microsoft shipped 5.4 and 5.4-pro on March 5, then dropped the mini and nano tiers on March 17. In the same breath, GPT-4o standard deployments start auto-upgrading on March 9 — forced retirement by end of month.

The GPT-5 Model Family: From Base to 5.4

The GPT-5 family is structured in generations, each bringing incremental improvements in reasoning quality, instruction following, and tool use reliability. GPT-5 base (August 2025) introduced the 400K context window with 272K input and 128K output capacity.

Key difference

GPT-5 models have a 400K context with parallel tool calling. The o-series has 200K context and processes tools sequentially.

GPT-5.1 (November 2025) added codex variants optimized for code generation. GPT-5.2 (December 2025) improved multi-step reasoning consistency. GPT-5.3 brought a new codex model in February 2026, and now GPT-5.4 represents the current state of the art.

GPT-5.4: What Is New

GPT-5.4 launched on March 5, 2026 alongside GPT-5.4-pro, with GPT-5.4-mini and GPT-5.4-nano arriving on March 17. The model is currently available in Global Standard deployment across East US 2, Sweden Central, South Central US, and Poland Central.
